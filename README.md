# AI Fullstack Developer Learning Internet From Zero

## Understanding modern software engineering from first principles.

This repository is not just another collection of notes.

It is an open book that documents my journey to understand how the Internet really works—from a single browser request to globally distributed systems.

Instead of memorizing technologies, this project focuses on answering one simple question:

"Why does this technology exist?"

Every chapter starts with the problem, explores the design decisions, and explains how modern engineering solves it.

Topics include:

• Networking
• HTTP
• Linux
• Nginx
• Node.js
• Databases
• Redis
• Docker
• Kubernetes
• AWS
• Distributed Systems
• System Design
• AI Engineering

This repository will continue to grow as I learn.
